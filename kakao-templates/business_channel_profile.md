## 카카오 채널 vs 솔라피 vs 알림톡 — 누가 뭘 하는지

```
[송준 대표]
   │
   ├─ ① 카카오톡 채널 (@데일리픽)         ← 회원이 친구 추가하는 곳, 광고성 친구톡 송출
   │      │
   │      └─ PFID 발급 (KA01PF...)         ← 솔라피에 등록
   │
   └─ ② 솔라피 (SOLAPI)                   ← 알림톡 발송 대행, API 제공
          │
          ├─ 발신 프로필: PFID 등록
          ├─ 알림톡 템플릿 5종 등록 → 카카오 검수 (1~3영업일)
          ├─ SMS/LMS 폴백 (알림톡 실패 시)
          └─ 친구톡 발송 (광고성, 채널 친구만 대상)
```
