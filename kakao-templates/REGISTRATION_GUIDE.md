# 카카오 알림톡 셋업 — 송준 대표(개인사업자) 기준

> 변호사 약관 검토는 생략하셨으므로, 카카오 검수에서 거절될 가능성이 가장 높은 **표현 규제**를 본 가이드에서 한 번 더 강제합니다. 검수 통과 후에 신고번호 자리(`#{license}`)만 채우면 즉시 발송 가능합니다.

---

## A. 한 페이지 셋업 절차 (총 3~5영업일)

| # | 단계 | 소요 | 비용 | 결과물 |
|---|------|------|------|--------|
| 1 | 사업자등록 (개인) | 즉시 ~ 1영업일 | 0원 | 사업자등록증 |
| 2 | 통신판매업 신고 | 1~3영업일 | 약 4만원/년 | 통신판매업 신고번호 |
| 3 | 카카오톡 채널 개설 ([center-pf.kakao.com](https://center-pf.kakao.com)) | 즉시 | 0원 | 채널 검색용 URL |
| 4 | 카카오 비즈니스 인증 (사업자등록증 업로드) | 1~3영업일 | 0원 | 비즈니스 채널 |
| 5 | 솔라피 가입 ([solapi.com](https://solapi.com)) → 본인 인증 → API 키 발급 | 즉시 | 0원 | API_KEY / API_SECRET |
| 6 | 솔라피에 카카오 채널 연동 (PFID 등록) | 즉시 ~ 1영업일 | 0원 | PFID |
| 7 | **알림톡 템플릿 5종 등록** (`templates.json` 그대로 입력) | 등록 즉시 / 검수 1~3영업일 | 0원 | 템플릿 ID 5개 |
| 8 | 유사투자자문업 신고 (금감원 전자민원창구) | 30일 | 0원 | 신고번호 → `#{license}` 채움 |
| 9 | 한국금융투자협회 보수교육 (연 1회 8시간) | 신고 후 | 약 15만원 | 수료증 |

## B. 검수 통과 전까지 본인 테스트하는 3가지 방법

| 방법 | 즉시 가능? | 비용 | 한계 |
|------|----------|------|------|
| **① 솔라피 SMS 발송** (본인 폰 → 본인 폰) | ✅ 솔라피 가입 즉시 | 건당 약 22원 | 90바이트 / LMS는 2,000자 — 알림톡과 발송 매체만 다르고 본문은 동일하게 미리보기 가능 |
| **② 친구톡 발송** (본인이 채널 친구 추가) | ✅ 채널 개설 + 친구 1명 추가만 하면 즉시 | 건당 약 14원 | 광고성으로 분류되어 21~익일 8시 발송 차단 — **운영 발송용 아님, 디자인 점검용**만 |
| **③ 알림톡 검수 통과 후 발송** | 1~3영업일 후 | 건당 약 8.5원 | 정식 운영 채널 (시간 제한 없음) |

> 추천 흐름: **D-3까지** ① SMS로 본문 가독성·길이 점검 → **D-2** 카카오에 템플릿 5종 동시 등록 → **D 당일** 검수 통과 → 본인 알림톡 수신 → 베타 100명 모집 시작.

테스트 코드는 [`tools/test_to_self.py`](../tools/test_to_self.py) 한 파일로 통합되어 있습니다.

```bash
# (1) 솔라피 SMS — 즉시 본인 폰 도착 (가독성 시뮬)
python tools/test_to_self.py sms 010-1234-5678 KR_DAILY

# (2) 친구톡 — 채널 친구 추가 후 본인 폰 도착 (디자인 점검)
python tools/test_to_self.py friendtalk 010-1234-5678 KR_DAILY

# (3) 알림톡 — 검수 통과 후 (정식)
python tools/test_to_self.py alimtalk 010-1234-5678 KR_DAILY
```

## C. 카카오 검수 통과를 높이는 8가지 원칙

알림톡(정보성)으로 인정받지 못하면 친구톡(광고성, 야간 차단)으로 강등됩니다. 등록 양식이 본 원칙을 모두 만족하도록 설계되어 있습니다.

1. **정보성 신호 명문화** — 본문 첫 줄에 "회원님이 신청하신 …" 문구를 반드시 포함
2. **변수만으로 본문 채우지 않기** — 변수 비율은 본문 길이의 50% 이하 (카카오 자동 거절 사유)
3. **광고 단어 금지** — 무료·특가·이벤트·한정·선착순·할인·증정·쿠폰
4. **자본시장법 금지 단어 금지** — 수익 보장·100% 적중·전문가 추천·반드시 오른다 (코드의 `FORBIDDEN_TERMS`)
5. **수신거부 안내 명시** — 본문 또는 버튼에 "수신거부" 또는 "정기결제 해지" 링크 포함
6. **문의처 표기** — 본문 또는 버튼에 고객센터 안내
7. **버튼 5개 이하** — 본 템플릿은 모두 2개 (상세 / 수신거부)
8. **이모지 최소화** — 알림톡은 이모지 1~2개까지 허용, 미장 신호등 🟢🟡🔴 만 사용

## D. 본인 폰으로 받기 위한 솔라피 환경변수

`.env` 파일을 프로젝트 루트에 생성:

```bash
# 솔라피 — solapi.com 가입 후 [개발 → API Key] 메뉴에서 발급
SOLAPI_API_KEY=NCSXXXXXXXXXXXX
SOLAPI_API_SECRET=XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
SOLAPI_SENDER=01012345678              # 본인 휴대폰 (사전 발신번호 등록 필수, 24h 소요)

# 카카오 비즈니스 채널 — solapi에 채널 연동 후 발급
ALIMTALK_PFID=KA01PF240429000000000000000000

# 검수 통과 후 카카오에서 부여하는 템플릿 ID — 등록 직후 빈 값으로 두고, 검수 통과 후 입력
ALIMTALK_TEMPLATE_ID_KR=
ALIMTALK_TEMPLATE_ID_US=
ALIMTALK_TEMPLATE_ID_PAYMENT_D1=
ALIMTALK_TEMPLATE_ID_WELCOME=
ALIMTALK_TEMPLATE_ID_TRIAL_END=

# 본인 테스트용 수신 번호 (생략 시 SOLAPI_SENDER로 발송)
TEST_RECIPIENT=01012345678
```

## E. 등록 양식을 솔라피에 입력하는 정확한 위치

1. 솔라피 로그인 → 좌측 메뉴 **[알림톡] → [템플릿 관리] → [템플릿 등록]**
2. `templates.json` 의 각 항목을 그대로 입력:
   - **템플릿명**: `name` 필드 (한글)
   - **카테고리**: `category_code` (정보성 분류 — 본 가이드는 모두 "회원관리/회원알림" 또는 "구매/이용내역")
   - **메시지 유형**: `message_type` — 기본형 / 부가정보형 / 채널추가형 / 복합형
   - **본문**: `body` 필드 그대로 복붙. `#{변수명}` 은 자동 인식
   - **버튼**: `buttons` 배열 그대로 입력
3. 등록 버튼 → 카카오 검수 신청 (1~3영업일)
4. 통과 시 **TPL_** 로 시작하는 템플릿 코드가 발급됨 → `.env` 의 `ALIMTALK_TEMPLATE_ID_*` 에 입력
